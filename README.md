# Weight Loss Tracker

A small, dependency-free browser app for tracking weight-loss progress.

## Features

- Goal/profile setup with start weight, target weight, and height
- Weight entry logging with date, weight, and notes
- Edit and delete existing entries
- Dashboard stats for current weight, total change, distance to goal, and BMI
- Responsive canvas chart for visual progress
- Unit toggle between kilograms and pounds
- JSON export and import
- Local browser storage persistence

## Run

Open `index.html` in a browser. No install step is required.

## Data

All tracker data is stored in the browser's `localStorage` under `weight-loss-tracker:v1`.
